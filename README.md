# BI Covid-19 - Dashboard com Dados das Vacinas Distribuídas e Aplicadas no Brasil
*Todos os dados apresentados são reais, obtidos a partir da base de dados do Ministério da Saúde, atualizados 2 vezes ao dia, as 10:00 e 15:00, 
com intuito de ter utilidade pública e apresentação do meu portifólio.*

## Análises Feitas Nesse Dashboard

### Os objetivos da análise de BI nesse projeto são para obter as seguintes informações:

- #### Nessa primeira página, são apresentados os dados referentes as vacinas distribuídas e aplicadas no Brasil por estado:
1. Velocímetro, informando o total de doses aplicadas x o total de doses distribuídas;
2. O mapa do Brasil, plotando os estados que receberam e aplicaram a vacina;
3. Valores absolutos das doses distribuídas e aplicadas das primeiras e segundas doses;

## Análises e Resultados Obtidos:

#### Com base nesse dashboard apresentado, podemos observar os seguintes dados e indicadores.

Que São Paulo é o estado que mais recebe e distribui doses de vacinas e que no Brasil, entre a distribuição e aplicação das vacinas, estamos em média na casa dos 80%.
<br />
Ainda temos 25,09% da população vacinada com a primeira dose e quando falamos em segunda dose, caímos para 11,27%.
